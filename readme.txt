== Description ==

Rotating Banner is a simple banner that can be rotated to any angle. It can be used as POS or call to actions, it allows for the angle of rotation, padding, margin and a shadow

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin zip file to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= Can I change the settings? =

Yes, not only are there settings for the angel, there are also setting for the background colour, padding and margins, shadow vertical and horizontal offset, shadow blur and shadow colour.

= What about border radius? =

Yes, although the border radius is currently using the Gutenberg experimental setting so it might break in the future but this will be fixed in due course.
